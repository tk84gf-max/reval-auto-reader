# ReVal 自動読み取り（不動産マイソク → 収益分析）

マイソク（販売図面・物件概要書のPDF/画像）をドロップするだけで、AIが数値を読み取り、
**収益指標（CF・CCR・返済比率・CF率）・積算・税引後CF・キャプテン基準の合否**を即算出し、
さらに **「想定／満室実数／現況実数」の3列比較**まで出すローカルアプリです。

計算エンジンは TC Capitals「InvestGate+（IG+）」を原型に作った ReVal。手入力での電卓としても使えます。

---

## 必要なもの
1. **Node.js**（無料）… https://nodejs.org の「LTS」を入れる（インストーラは全部Nextで可）
2. **Anthropic APIキー**（マイソクのAI読み取りに使用）… https://console.anthropic.com で取得
   - 使った分だけの従量課金。**各自のキー＝各自の負担**（マイソク1枚あたり数円程度）
   - 手入力だけで使うなら不要（電卓としてはキー無しでも動きます）

## セットアップ（初回だけ・5分）
1. このフォルダ一式をダウンロード（ZIP）または `git clone`
2. **`初回設定_APIキー.bat`** をダブルクリック → 開いたメモ帳に APIキー（`sk-ant-...`）を貼って保存
   - 鍵は `C:\Users\<あなた>\.config\reval\apikey.txt` に保存されます（このフォルダには入りません＝安全）
3. **`ReVal自動読み取りを起動.bat`** をダブルクリック → ブラウザが開く（http://localhost:5178）

## 使い方
- マイソクをドロップ（複数まとめてOK）／スクショを Ctrl+V → 物件カードが増える
- 読み取った数字を確認・修正（自動で再計算）
- 「📎 詳細資料を追加」にレントロール/ランニングコストを入れると、満室実数・現況実数も算出
- 「物件価格」を動かすと、賃料を固定したまま表面利回りが変わる（価格交渉シミュレーション）
- 黒い画面は開いたままに（閉じるとアプリ停止・落ちても自動再起動）

## カスタムしたい人へ
- **電卓部分（ReVal.html）は1ファイル完結**。メモ帳/VSCodeで開いて、標準費率・キャプテン基準の数値などを自由に書き換えられます。
- GitHubで **Fork** すれば自分専用版を持てます。本体が更新されたら `pull` で取り込めます。

## 安全・前提
- APIキーはこのフォルダ／GitHubには絶対入りません（`~/.config/reval/` に保存）。
- 個人の物件データはあなたのPC内（`analysis/`）にのみ生成され、リポジトリには含まれません（`.gitignore`済み）。
- 本ツールは投資判断の補助であり、最終判断・自己責任でご利用ください。キャプテン基準は一例です。

## 構成（技術メモ）
- `server.js` … ローカルサーバー（Node標準のみ・npm不要）。`/extract`でAI読み取り、`/calculate`で計算。
- `reval_core.js` … 共通コア（Vision抽出＋エンジン呼び出し）
- `index.html` … 物件カード・ワークスペースUI
- `ReVal.html` … 計算エンジン本体（電卓・3列比較・印刷用想定収支）
- `scripts/reval_calc.js` `scripts/prefill_reval.js` … 計算エンジン（Node移植版）
